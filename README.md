  Github'da repository nasıl silinir?
Github'da yarattığım bir repository'yi silerek visual studio üzerinden tekrar yaratmam gerekti ve 10 dakika aynı ekranlarda bir remove repository düğmesi arayarak zaman kaybettim, ben yandım başkaları benim kadar zaman kaybetmesinler diyerek adım adım github'dan repo kaldırmanız için neler yapmanız gerektiğini yazıyorum:

1. Github'ı açın ve giriş yapın

2. Sağ üst taraftaki profil fotoğrafınızın sağındali oka basın ve "your repositories" linkine tıklayın

3. silmek istediğiniz repository'ye tıklayın

4. Üst taraftaki menüden settings'e tıklayın

5. Açılan sayfayı en aşağı kadar kaydırın ve danger zone (tehlike alanı) olarak adlandırılmış kırmızı border ile çevrili alandan "delete this repository" butonuna tıklayın. Alternatif olarak bu alandan reponuzu başkasına transfer edebilir, private ise public yapabilir ya da arşivleyebilirsiniz.

6. delete this repository butonuna kazara basanlar da olabilir diyerek github size bir popup pencere gösterecek ve eminseniz bu kutucuğa repository'nizin adını yazın ve alt taraftaki "I understand the censequences, delete this repository" (sonuçlarının farkındayım, repo'yu sil) butonuna basın.

        https://www.reitix.com/merak/github-da-repository-nasil-silinir/de04df36
